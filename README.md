

# Java Core Training

Welcome to my **Java Core Training** repository. This repository contains the programs, examples, and practice exercises completed during my Java Core training.

## 📖 About

The purpose of this repository is to learn and practice the fundamentals of Java programming. It includes basic concepts, hands-on examples, and small programs that help build a strong foundation in Java.

## 🚀 Topics Covered

- Introduction to Java
- Java Installation & JDK Setup
- Java Program Structure
- Variables and Data Types
- Operators
- Input and Output
- Comments in Java
- Type Casting
- Control Statements
  - if, if-else
  - switch
  - Loops (for, while, do-while)
- Arrays
- Methods
- Object-Oriented Programming (OOP)
  - Classes and Objects
  - Constructors
  - Inheritance
  - Polymorphism
  - Abstraction
  - Encapsulation
- String Handling
- Exception Handling
- Packages
- Collections Framework
- File Handling
- Multithreading
- Java 8 Features (Basics)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/siddhant256/FUEL-Full-Stack-Java-Training.git
```

2. Open the project in your preferred Java IDE.

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```
